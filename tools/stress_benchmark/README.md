# Праверка вызначэння націскаў

- stress_benchmark.py - скрыпт для праверкі якасці вызначэння націскаў.
- data/common_voice.txt - 29 найбольш частотных амаграфаў кожны з прыкладна 100 узорамі
- data/zascienak_Malinauka.txt - першая частка кнігі "Засценак Малінаўка"

У абодвух датасэтах націскі пазначаныя носьбітамі мовы ўручную.

Праверка паказвае наступныя лічбы правільнасці вызначэння націскаў:

    StressStat :
        - common_voice.txt: 0.86
        - zascienak_Malinauka.txt: 0.92

    StressLLM + Gemini 3.1 Flash Lite (thinking=None) :
        - common_voice.txt: 0.92
        - zascienak_Malinauka.txt: 0.88

    StressLLM + Gemma 4 26B (thinking=None) :
        - common_voice.txt: 0.78
        - zascienak_Malinauka.txt: 

Ёсць словы, якія патрабуюць дадаванне значэння з Тлумачальнага слоўніку для правільнага вызначэння націска, бо граматычных характарыстык недастаткова.
Напрыклад, словы: зараз, лісце, нарадзіцца, гадзіна, хапала, вады, мука, etc.

Таксама, data/common_voice.txt трэба перагледзіць каб выключыць сказы, ў якіх немагчыма правільна вызначыць націск.
